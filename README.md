# Portfolio

A personal portfolio website built with Next.js to showcase projects, education, platforms, achievements, and contact details in a clean responsive layout.

## Features

- Project gallery with images, descriptions, and tech stack tags.
- About, Education, Platforms, and Contact pages.
- Responsive layout for desktop and mobile.
- Direct contact option through email instead of a backend form.
- Built with help from AI tools during development and content refinement.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- JavaScript / JSX
- TypeScript config support

## Getting Started

Live Website: https://welcome-to-kresha-portfolio.vercel.app/

## Project Structure

- `app/` route pages and global styles
- `components/` reusable UI components
- `public/` static assets and project images

## Deployment

This project can be deployed to Vercel or any platform that supports Next.js.

Use the Contact page to open your email app or copy the email address directly.
